# Search Engine (From Scratch)

## Goal
Build a search engine from first principles to deeply understand
information retrieval, indexing, and ranking. Overall, to understand systems better.


## Non-goals (for now)
- Web crawling
- UI
- Databases
- Frameworks

## Current Scope
- Local .txt documents
- Command-line interface
- Inverted index
- AND queries


## Design choice

tokenizer.js (v1)

-- tokens include lowercase letters and numbers.
-- why: docs file expected to contain technical writing.
-- keep everything else as a separators
